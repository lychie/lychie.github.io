There is no getter for property named 'xx' in 'class java.lang.String'

作者：範宗雲 来源：原创 发布时间：2015-05-09 归档：exceptions

mybatis 环境 
		    	<select id="find" resultType="Person">
		    	  select * from person where address like ${address}
		    	</select>
		    
		    	public interface PersonMapper {
					
					List<Person> find(String address);
					
				}

如果 sql 参数是 #{address}, 执行不会报错。这里使用的参数是 ${address}, 报了上面的错误。既然报的是 There is no getter for property named 'address' in 'class java.lang.String', 那么解决办法很简单, sql 参数类型不用 string 就是了, 改成 map 或 pojo 类型, 如 map.put("address", "xxxxx") 即可。或者, 你还可以这样做 ： 
		    	public interface PersonMapper {
					
					List<Person> find(@Param("address") String address);
					
				}
通过 @Param 标注, 指明参数的名称, 这样问题也可以解决。
